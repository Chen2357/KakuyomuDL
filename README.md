# KakuyomuDL

KakuyomuDL is a command line interface built with Swift for downloading novels from [Kakuyomu](https://kakuyomu.jp/) and converting them to EPUB with vertical text.

Style of the generated EPUB is based on the EPUB generated by [https://t2aki.doncha.net/?id=2778](https://t2aki.doncha.net/?id=2778).

**Uses that violate Kakuyomu's terms of service are strictly prohibited.
Please use this tool responsibly and respect the rights of the authors.**

## Basic Usage

1. Clone the repository.
2. Identify the work ID from the Kakuyomu URL. For example, if the URL is `https://kakuyomu.jp/works/1234567890123456789`, the work ID is `1234567890123456789`.
3. Build and run with Swift Package Manager.
   ```
   swift run KakuyomuDL <work_id> <output_path> --cache <cache_path>
   ```

It is highly recommended to specify a cache path.
The cache will store the downloaded HTML files, so you can re-run the command without downloading again.

To see the help message, run:
```
swift run KakuyomuDL --help
```

## Special Transformations

Special transformations are applied on a work-by-work basis.
They directly modify the content of the work to improve formatting.
The transformations are defined in [SpecialTransformation.swift](Sources/KakuyomuDL/SpecialTransformation.swift), and you can add your own transformations.
